Application Flow Chart

The following graphic illustrates how data flows throughout the system:

  1. The index.php serves as the front controller which initializes the base resources needed to run Code Igniter.
  2. Security. Before anything of note takes place, the HTTP request is filtered for security.
  3. The Router examines the HTTP request to determine which application controller should be used.
  4. The Controller loads the core libraries, plugins, and helpers needed to process the specific request.
  5. The finalized View (web page) is rendered then sent to the web browser to be seen.
