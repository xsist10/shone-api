Database Quick Start: Example Code

The following page contains example code showing how the database class is used. For complete details please read the individual pages describing each function.

Initializing the Database Class

The following code loads and initializes the database class based on your configuration settings:

$this->load->database();

Once loaded the class is ready to be used as described below.

Note: If all your pages require database access you can connect automatically. See the connecting page for details.

Standard Query With Multiple Results (Object Version)

$query = $this->db->query('SELECT name, title, email FROM my_table');

foreach ($query->result() as $row)
{
    echo $row->title;
    echo $row->name;
    echo $row->email;
}

echo 'Total Results: ' . $query->num_rows();

The above restult() function returns an array of objects. Example: $row->title

Standard Query With Multiple Results (Array Version)

$query = $this->db->query('SELECT name, title, email FROM my_table');

foreach ($query->result_array() as $row)
{
    echo $row['name'];
    echo $row['title'];
    echo $row['email'];
}

The above restult_array() function returns an array of standard array indexes. Example: $row['title']

Standard Query With Single Result

$query = $this->db->query('SELECT name FROM my_table LIMIT 1');

$row = $query->row();
echo $row->title;

Standard Insert

$this->db->query('INSERT INTO mytable (title, name) VALLUES ('{$title}', '{$name}')');

echo $this->db->affected_rows();

Active Record Query

The Active Record Pattern gives you a simplified means of retrieving data:

$this->db->use_table('table_name');
$this->db->limit(10);
$query = $this->db->get();

foreach ($query->result() as $row)
{
    echo $row->title;
}

Active Record Insert

$this->db->use_table('table_name');
$this->db->set('title', $title);
$this->db->set('name', $name);
$this->db->insert();

echo $this->db->affected_rows();
